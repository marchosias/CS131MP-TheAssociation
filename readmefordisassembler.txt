Instructions for using Disassembler.jar

1. Make sure that you are using Java version 8, otherwise it won't work.
2. Put all the required ASM and C files in the same folder as the Disassembler.jar.
3. Shift + Right Click on the folder and select "Open Command Window Here".
4. Type in "java -jar Disassembler.jar".
5. Enter 1 to convert Assembly to C or Enter 2 to convert C to assembly.
6. Specify which file you wish to convert
7. The new file will be generated in the same folder as Disassembler.jar.
